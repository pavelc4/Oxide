## About Oxide
Oxide is a reliable ADB client app designed to simplify the comprehensive management of your Android devices. Manage apps, browse and transfer files, monitor device performance in real time, and even flash partitions all within a single, simple, and intuitive interface.

## Features
- **Device Management** — List, inspect, and monitor connected ADB/Fastboot devices
- **App Manager** — List, install, uninstall, clear, and batch-manage packages
- **File Explorer** — Push, pull, browse, and manage device filesystem
- **Performance Monitor** — Real-time CPU, RAM, battery
- **Wireless ADB** — Connect/disconnect devices over TCP/IP
- **Fastboot Flashing** — Flash partitions, manage slots, OEM commands, sideload
- **Logcat Viewer** — Stream and filter device logs in real-time
- **Audit Log** — Persistent JSON operation log (opt-in)
